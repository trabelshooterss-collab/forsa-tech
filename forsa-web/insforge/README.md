# تفعيل ميزة المزادات (InsForge)

الويب جاهز للعرض، لكن المزادات تحتاج تفعيل جداولها مرة واحدة في قاعدة البيانات.

1. افتحي محرر SQL في مشروع InsForge وشغّلي ملف `insforge/auctions.sql`.
2. أنشئي Bucket في التخزين باسم `auction-images` (علشان صور المزادات).
3. بعد كده صفحة `المزادات` هتعرض بيانات حقيقية.
4. بعد كده صفحة `إنشاء مزاد` هتنشر مزاد فعلي.
5. بعد كده صفحة المزاد هتقبل مزايدات حقيقية.
