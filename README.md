# Forsa-Tech - منصة فرصة التقنية

## نظرة عامة
منصة Forsa-Tech هي منصة عربية متكاملة للفرص التقنية، تهدف إلى ربط الشركات والمطورين والباحثين عن عمل في مجال التكنولوجيا.

## المميزات الرئيسية
1. نظام إدارة الفرص التقنية
2. بوابة للمطورين والباحثين عن عمل
3. نظام إدارة الشركات
4. نظام التوظيف والترشيحات
5. نظام تقييم المهارات
6. نظام التعليم والتدريب

## التقنيات المستخدمة
- Frontend: React.js
- Backend: Node.js
- Database: PostgreSQL
- API: RESTful API
- Authentication: JWT
- Deployment: Docker

## البنية الأساسية للمشروع
```
FORSA/
├── forsa-web/          # واجهة المستخدم الرئيسية
├── forsa-api/          # API الخلفي
├── forsa-admin/        # لوحة التحكم
├── forsa-mobile/       # التطبيق المحمول
├── forsa-docs/         # الوثائق
└── docker/             # ملفات Docker
```
